# Boilerplate for Vue, Vuex and Vue-Router

## How to use

1. download & install

```bash
$ git clone <this.repository>

$ cd <this.repository>

## Change name and author, description, etc... in package.json

$ npm install

```

2. develop

```bash

# coding with gulp

$ npx gulp

## display color log and message

$ npx gulp --color


# publish files to dist

$ npx gulp build

```
